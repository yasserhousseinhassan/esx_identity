## About This Version

This is a modern redesign of the classic esx_identity resource for FiveM. The redesign features a completely updated user interface with modern UI/UX design principles, smoother animations, and an improved user experience.

### What's New in This Redesign

- Modern UI - Fresh, redesigned interface with contemporary styling
- Smooth Animations - Enhanced transitions and micro-interactions
- Better UX - Improved user flow and experience
- Modern Design - Clean aesthetic with better visual hierarchy
- Optimized Code - Refactored for better performance

---

## Description

A Core Resource that Allows the player to Pick their characters, Name, Gender, Height and Date-of-birth.
